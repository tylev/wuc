A simple real-time Bitcoin price widget

What is WeUseCoins Bitcoin Price Widget?

WeUseCoins Bitcoin Price widget is an open-source, real-time Bitcoin price widget for your website or blog. It is easily embeddable with 2 lines of code and offers a beautifully minimal design.

Which technologies does the WeUseCoins Bitcoin price widget use?

WeUseCoins Bitcoin price widget uses the Bitcoin Open Data Set for real-time price updates.

Where can I use the WeUseCoins Bitcoin price widget?

WeUseCoins Bitcoin Price widget is completely free and can be used on any website or blog without restriction. The source code to WeUseCoins is published on GitHub so please send a pull request when you're ready to contribute and is under the MIT license, so feel free to improve, adapt, or redistribute it as you like.

Where does the data come from?

Bitcoin price data is provided by Coinbase.
